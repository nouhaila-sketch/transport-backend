# Système de Gestion de Transport Urbain

Projet de réimplémentation des cas d'utilisation (Bus, Lignes, Horaires) développé dans le cadre du module **Spring**.

## Architecture du Projet
* **Backend :** Développé avec Java / Spring Boot (REST API exposant les données au format JSON).
* **Frontend :** Interface moderne et responsive développée pour consommer les services du backend.

## Comment lancer le projet :
1. **Backend :** Ouvrir le dossier `TRANSPORT` avec votre IDE et lancer l'application Spring Boot (`mvn spring-boot:run`).
2. **Frontend :** Ouvrir le fichier `index.html` situé dans le dossier `frontend` directement sur votre navigateur web.
